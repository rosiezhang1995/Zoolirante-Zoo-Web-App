# 🦁 Zoolirante Web Application

Zoolirante is a zoo web application designed to enhance the customer experience by making it easier for users to explore animals, browse events, and interact with personalised zoo content. The platform includes features such as an animal list, events list, and the ability to favourite animals and events.

---

## 📖 Project Overview

This project was developed as a **team-based product release** using an **Agile workflow**. The goal was to create an engaging, user-friendly, and interactive zoo platform that supports both visitor browsing and booking-related activities.

---

## ✨ Key User Features

- 🐾 Animal list and animal detail pages  
- 🎉 Events list and event-related functionality  
- ❤️ Favourite animals and saved events  
- 🎟️ Ticket purchasing  
- 🔔 Notification system  
- 💳 Payment method saving  
- 🗺️ Membership, merchandise, and zoo map support

---

## 🛠️ Admin Features

- ➕ Add/Edit Animals and Events  
- 📋 Booking management  
- 👤 User account management  

---

## 💻 Tech Stack

- **ASP.NET** 
- **JavaScript**  
- **Tailwind CSS**
